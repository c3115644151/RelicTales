# RelicTales（考古物语）- MVP 设计文档

**版本：** v2.0 (2026/04/28)
**核心哲学：** 原生增强 (Vanilla+)、少即是多、脚踏实地

---

## 一、核心概念

本模组作为原版考古（Archeology）的自然延伸，将"可疑方块"注入玩家熟悉的原版遗迹中，引导重新探索。

### 核心循环
```
前往遗迹 → 发现可疑方块 → 原版刷子刷取 → 获得物品
```

### 设计哲学

少即是多。不与原版 RPG 化趋势同流。

考古产出不应该是数值装备，而应该是"视角切换器"——让玩家看到以前看不到的东西，而不是让玩家变得更强。
遗迹注入的最终目的是让玩家**慢下来**，而不是给玩家一条新的打怪练级路线。

---

## 二、考古方块系统

所有可疑方块继承原版 `BrushableBlock`，复用原生刷取机制。

| 原版遗迹 | 注入方块 | 当前状态 |
|---|---|---|
| **丛林神殿** | `suspicious_mossy_cobblestone` | ✅ 已注入（Mixin） |
| **沙漠神殿** | `suspicious_chiseled_sandstone` | ✅ 已注入（Mixin） |
| **要塞** | `suspicious_cracked_stone_bricks` / `suspicious_mossy_stone_bricks` / `suspicious_mossy_cobblestone` | ✅ 已注入（Mixin 概率系统） |
| **下界要塞** | `suspicious_nether_bricks` | ✅ 已注入（Mixin 空气暴露检测） |
| **末地城** | `suspicious_purpur_block` | ⏳ 待设计 |

### 工具
玩家直接使用原版刷子（`minecraft:brush`）参与考古，无需合成任何自定义工具。

---

## 三、已废弃系统

- ❌ 诅咒系统（完全移除）
- ❌ 沉积岩地层生成（不改变世界 Y 轴结构）
- ❌ 复杂錾子交互（无自定义工具分级）
- ❌ 丛林猎羽符（数值化遗物，不符合新设计哲学）
- ❌ 太阳神徽章（同上）
- ❌ 凋零护符 / 虚空记录碎片 / 亚特兰蒂斯陶罐之心（同上，全部清理）

## 五、配置设计原则

所有可变数值（掉落概率、刷取次数等）必须进入 `config/relictales-common.toml`，
硬核逻辑留在 `content/` 层。
