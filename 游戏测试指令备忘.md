# 游戏测试指令备忘

## 常用辅助指令

| 用途 | 指令 |
|---|---|
| **永久夜视** | `/effect give @p minecraft:night_vision infinite 0 true` |
| 切换创造模式 | `/gamemode creative` |
| 切换生存模式 | `/gamemode survival` |
| 保持物品栏 | `/gamerule keepInventory true` |
| 清除怪物 | `/kill @e[type=!minecraft:player]` |
| 晴天 | `/weather clear` |
| 设为白天 | `/time set day` |
| 给予刷子 | `/give @p minecraft:brush` |

## 遗迹定位

| 用途 | 指令 |
|---|---|
| 定位要塞 | `/locate structure minecraft:stronghold` |
| 定位沙漠神殿 | `/locate structure minecraft:desert_pyramid` |
| 定位丛林神殿 | `/locate structure minecraft:jungle_temple` |
| 定位下界堡垒 | `/locate structure minecraft:fortress` |
| 传送到坐标 | `/tp @p <x> <y> <z>` |
