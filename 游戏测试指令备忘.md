# 游戏测试指令备忘

## 常用辅助指令

| 用途 | 指令 |
|---|---|
| **永久夜视** | `/effect give @p minecraft:night_vision infinite 0 true` |
| 切换创造模式 | `/gamemode creative` |
| 切换生存模式 | `/gamemode survival` |
| 保持物品栏 | `/gamerule keepInventory true` |
| 清除怪物 | `/kill @e[type=!minecraft:player]` |
| 晴天 | `/weather clear` |
| 设为白天 | `/time set day` |

## 遗迹定位

| 用途 | 指令 |
|---|---|
| 定位要塞 | `/locate structure minecraft:stronghold` |
| 定位沙漠神殿 | `/locate structure minecraft:desert_pyramid` |
| 定位丛林神殿 | `/locate structure minecraft:jungle_temple` |
| 定位下界堡垒 | `/locate structure minecraft:fortress` |
| 定位末地城 | `/locate structure minecraft:end_city` |

## 末地/末地城传送

末地城只在末地外岛生成（距中心 >1000 格）。

| 用途 | 指令 |
|---|---|
| **一链传送到末地城** | 先传送到末地后再定位+传送 |
| 传送到末地 | `/execute in minecraft:the_end run teleport @p 0 64 0` |
| 定位末地城 | `/locate structure minecraft:end_city` |
| 传送到末地城坐标 | `/teleport @p <x> <y> <z>` |

> **快捷三步**（复制后依次执行）：
> 1. `/execute in minecraft:the_end run teleport @p 0 64 0`
> 2. `/locate structure minecraft:end_city` → 记下坐标
> 3. `/teleport @p <x> <y> <z>`
