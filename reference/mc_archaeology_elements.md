## 三、原版盔甲纹饰系统（20 种）

> 盔甲纹饰本质上是"古代锻造工艺传承"，天然属于考古范畴。
> 目前仅 4 种来自考古（Trail Ruins），其余来自其他结构。

| 纹饰 | 来源结构 | 考古状态 |
|---|---|---|
| Host / Raiser / Shaper / Wayfinder | Trail Ruins（考古） | ✅ 已接入 |
| Coast / Dune / Eye / Flow 等 4 种 | 沙漠/丛林/海洋/下界 | ❌ 未接入考古 |
| Wild / Tide / Sentry / Vex 等 4 种 | 远古城市 / 沉船 / 掠夺者 / 林地 | ❌ 未接入考古 |
| Silence / Snout / Spire / Rib 等 4 种 | Ancient City / 堡垒 / 末地 | ❌ 未接入考古 |
| Bolt | 试炼密室 Trial Chambers | ❌ 未接入考古 |

> 潜在方向：既然所有盔甲纹饰都是古代文明的产物，理论上都可以通过考古刷取获得。目前断裂的原因是纹饰是锻造模板形式（消耗品），而考古产出模式是"从土里刷出来"。

## 四、村民职业系统（13 种，26.1）

> 26.1 村民交易已改为数据驱动（`villager_trade` + `trade_set` JSON），这意味着我们可以用纯 JSON 添加新职业和交易，无需写 Java。

### 4.1 现有职业

| 职业 | 工作站 | 与考古交集 |
|---|---|---|
| 制图师 | 制图台 | 出售探险家地图 → 可扩展为"考古地图"（指向遗迹） |
| 石匠 | 切石机 | 处理石头/陶瓦 → 可扩展为"修复破损陶器" |
| 流浪商人 | （无工作站） | 沙漠装束，天然线索提供者 + 嗅探兽蛋贩卖 |
| 牧师 | 酿造台 | 可处理可疑炖菜相关交易 |
| 皮匠 | 炼药锅 | 可能涉及文物清洗/保护？ |

### 4.2 新职业方案

> 表意不要过于隆重，和其他村民平等。

**建议名称：**
- 中文：**探宝人**（和 渔夫、农民、牧羊人 同一气质）
- 英文：**Troweler**（trowel 泥铲 + -er，和 builder, farmer 同构）

**工作站方案：**
- 遴选台 / 筛选台（Sifting Table）：合成配方沿用刷子的元素（木棍 + 线）
- 或直接用**切石机**（和石匠共用工作站，但职业不同）

## 六、平衡性备注

### 埋藏宝藏现存问题

| 维度 | 说明 |
|---|---|
| 问题 | 沉船藏宝图 → 埋藏点 → 一次挖出铁锭x4 + 金锭x4 + 钻石x1 |
| 代价 | 仅需找到一艘沉船 + 挖几下沙子 |
| 与考古的关系 | 这是"挖宝"而非"刷宝"，不在同一体验链上 |
| 建议态度 | 不必调整原版数值。我们的考古系统需要提供一个"更稳定、更有成长性、更好玩"的选择，用系统质量竞争，不是数值竞争 |

## 七、考古生态位对比图

```
高风险高回报 ──── 试炼密室（战斗RPG）
                      │
                      │
                      │
  低风险线性回报 ── 原版遗迹考古（我们）
                      │
                      │
                      │
  低风险随机回报 ── 钓鱼
                      │
                      │
                      │
  高风险不确定 ──── 下矿
```

考古应该位于"低风险（不需要战斗装备）+ 成长性回报（靠知识和熟练度）"的生态位，和钓鱼平行，但回报方向不同——钓鱼给食物 + 宝藏，考古给知识 + 便利 + 装饰。
